#### Install packages
```sh
yarn install
```

#### Build Process
```sh
yarn build
```

#### Dev Process
```sh
yarn dev
```

#### Check Eslint
```sh
yarn lint
```

### define ENV
This project are used global .env configurations.
Please config env on `./.env.development` and `./.env.production`.
